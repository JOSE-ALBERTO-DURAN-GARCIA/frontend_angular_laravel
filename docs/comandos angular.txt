1. genrar el componente inicio
ng generate component inicio
ng g c inicio

2. Generar un modulo (auth)
ng g m auth --routing


3. Generar un modulo (admin)
ng g m admin --routing

4. Generar los componentes login y registro en el modulo auth
ng g c auth/components/login
ng g c auth/components/registro

5. Generar los componentes perfil en el modulo admin
ng g c admin/components/perfil
